## 問題 19: 自動化農場巡邏

### 問題內容
請告訴我：
1. 執行完整農場巡邏，總共收穫了多少次？
2. 巡邏後乾草總數量 = ?
3. 無人機最終位置 = ?

### 對應程式碼
```python
# 問題 19: 自動化農場巡邏
print("=== 問題 19: 自動化農場巡邏 ===")
print("開始完整農場巡邏...")
print("起始位置: (" + str(get_pos_x()) + ", " + str(get_pos_y()) + ")")
print("起始乾草數量: " + str(num_items(Items.Hay)))

harvest_count = 0
farm_size = get_world_size()

# S型巡邏整個農場
for y in range(farm_size):
    for x in range(farm_size):
        # 嘗試收穫
        if can_harvest():
            harvest()
            harvest_count += 1
        
        # 移動到下一格
        if x < farm_size - 1:
            if y % 2 == 0:
                move(East)
            else:
                move(West)
    
    # 移動到下一行
    if y < farm_size - 1:
        move(North)

print("巡邏完成！")
print("總收穫次數: " + str(harvest_count))
print("最終乾草數量: " + str(num_items(Items.Hay)))
print("最終位置: (" + str(get_pos_x()) + ", " + str(get_pos_y()) + ")")
```

## 問題 20: 智能澆水策略

### 問題內容
請告訴我：
1. 找到多少個需要澆水的地塊？
2. 澆水後平均水分等級 = ?
3. 消耗了多少個水桶？

### 對應程式碼
```python
# 問題 20: 智能澆水策略
print("=== 問題 20: 智能澆水策略 ===")
print("開始智能澆水...")
print("起始水桶數量: " + str(num_items(Items.Water)))

watered_count = 0
total_water_level = 0
watered_tiles = 0
farm_size = get_world_size()

# 巡邏農場尋找需要澆水的地塊
for y in range(farm_size):
    for x in range(farm_size):
        current_water = get_water()
        
        # 如果水分低於0.5，就澆水
        if current_water < 0.5:
            if use_item(Items.Water):
                watered_count += 1
                total_water_level += get_water()
                watered_tiles += 1
        
        # 移動到下一格
        if x < farm_size - 1:
            if y % 2 == 0:
                move(East)
            else:
                move(West)
    
    # 移動到下一行
    if y < farm_size - 1:
        move(North)

print("澆水完成！")
print("澆水地塊數量: " + str(watered_tiles))
if watered_tiles > 0:
    print("平均水分等級: " + str(total_water_level / watered_tiles))
print("消耗水桶數量: " + str(watered_count))
print("剩餘水桶數量: " + str(num_items(Items.Water)))
```

## 問題 21: 植物種植優化

### 問題內容
請告訴我：
1. 種植了多少個灌木？
2. 種植後木材總數量 = ?
3. 種植成功率 = ? (成功數/嘗試數)

### 對應程式碼
```python
# 問題 21: 植物種植優化
print("=== 問題 21: 植物種植優化 ===")
print("開始植物種植優化...")
print("起始木材數量: " + str(num_items(Items.Wood)))

planted_count = 0
attempted_count = 0
farm_size = get_world_size()

# 巡邏農場尋找可種植的地塊
for y in range(farm_size):
    for x in range(farm_size):
        # 檢查是否可以種植
        if get_entity() == None:  # 沒有實體
            attempted_count += 1
            if plant(Entities.Bush):
                planted_count += 1
        
        # 移動到下一格
        if x < farm_size - 1:
            if y % 2 == 0:
                move(East)
            else:
                move(West)
    
    # 移動到下一行
    if y < farm_size - 1:
        move(North)

print("種植完成！")
print("種植灌木數量: " + str(planted_count))
print("種植嘗試次數: " + str(attempted_count))
if attempted_count > 0:
    print("種植成功率: " + str(planted_count / attempted_count * 100) + "%")
print("最終木材數量: " + str(num_items(Items.Wood)))
```

## 問題 22: 效率測試

### 問題內容
請告訴我：
1. 執行完整農場管理需要多少時間？
2. 每分鐘收穫效率 = ? (收穫次數/時間)
3. 資源消耗效率 = ? (水桶消耗/時間)

### 對應程式碼
```python
# 問題 22: 效率測試
print("=== 問題 22: 效率測試 ===")
print("開始效率測試...")

start_time = get_time()
start_water = num_items(Items.Water)
start_hay = num_items(Items.Hay)

harvest_count = 0
water_count = 0
farm_size = get_world_size()

# 執行完整農場管理
for y in range(farm_size):
    for x in range(farm_size):
        # 收穫
        if can_harvest():
            harvest()
            harvest_count += 1
        
        # 澆水
        if get_water() < 0.5:
            if use_item(Items.Water):
                water_count += 1
        
        # 移動
        if x < farm_size - 1:
            if y % 2 == 0:
                move(East)
            else:
                move(West)
    
    if y < farm_size - 1:
        move(North)

end_time = get_time()
total_time = end_time - start_time

print("效率測試完成！")
print("總執行時間: " + str(total_time) + " 秒")
print("收穫次數: " + str(harvest_count))
print("澆水次數: " + str(water_count))
print("每分鐘收穫效率: " + str(harvest_count / (total_time / 60)) + " 次/分鐘")
print("每分鐘澆水效率: " + str(water_count / (total_time / 60)) + " 次/分鐘")
```

## 問題 24: 自動化循環測試

### 問題內容
請告訴我：
1. 執行3輪完整農場管理後總收穫 = ?
2. 平均每輪收穫效率 = ?
3. 資源消耗趨勢 = ?

### 對應程式碼
```python
# 問題 24: 自動化循環測試
print("=== 問題 24: 自動化循環測試 ===")
print("開始自動化循環測試...")

total_harvests = 0
total_water_used = 0
farm_size = get_world_size()

for round_num in range(3):
    print("執行第 " + str(round_num + 1) + " 輪...")
    round_harvests = 0
    round_water = 0
    
    # 一輪完整農場管理
    for y in range(farm_size):
        for x in range(farm_size):
            # 收穫
            if can_harvest():
                harvest()
                round_harvests += 1
            
            # 澆水
            if get_water() < 0.5:
                if use_item(Items.Water):
                    round_water += 1
            
            # 移動
            if x < farm_size - 1:
                if y % 2 == 0:
                    move(East)
                else:
                    move(West)
        
        if y < farm_size - 1:
            move(North)
    
    total_harvests += round_harvests
    total_water_used += round_water
    print("第 " + str(round_num + 1) + " 輪收穫: " + str(round_harvests) + " 次")

print("循環測試完成！")
print("總收穫次數: " + str(total_harvests))
print("平均每輪收穫: " + str(total_harvests / 3))
print("總消耗水桶: " + str(total_water_used))
print("平均每輪水桶: " + str(total_water_used / 3))
```
