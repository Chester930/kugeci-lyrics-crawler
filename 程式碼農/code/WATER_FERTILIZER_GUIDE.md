## 🌱 肥料系统详解

### 肥料效果
```python
use_item(Items.Fertilizer)
→ 减少 2 秒生长时间
→ 植物被感染（Infected）
```

### 感染机制
```
感染效果：
- 收获时 50% 产量 → Weird_Substance
- 实际产量减半

治疗方法：
use_item(Items.Weird_Substance)
→ 切换感染状态（自己 + 相邻植物）
```

---

## 🧮 肥料收益分析

### 场景 1：不使用肥料
```
生长时间：100 秒
产量：100%
总价值：100
```

### 场景 2：使用肥料，不治疗
```
生长时间：50 秒（-50%）
产量：50%（感染）
总价值：50

但在相同时间内可以种植 2 次：
总价值：50 × 2 = 100
```

**结论：不治疗时，收益持平** ⚖️

---

### 场景 3：使用肥料 + 治疗
```
生长时间：50 秒（-50%）
产量：100%（治疗后）
总价值：100

在相同时间内可以种植 2 次：
总价值：100 × 2 = 200
```

**结论：治疗后，收益翻倍！** 🚀
